Git Add:Permite agregar contenido al directorio de trabajo.
Git Commit: Es para agregar comentarios de los cambios en los repositorios.
Git Push: Es para guardar en el repositorio y hacer efectiva la sincronización del local al remoto.
Git Fetch: Es el comando que permite recuperar la ultima actualización.
Git Checkout : Permite desplazarse entre las ramas creadas.
Git Merge: Permite fusionar una o más ramas dentro de la rama que se tiene activa.
